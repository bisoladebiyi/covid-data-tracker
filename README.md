## Live Link:

Access the site at (https://covid-data-tracker-assessment.netlify.app)

![Landing Page](/src/assets/images/landing.png)

## Set Up 

In the project directory, you can run:

#### Installation

`npm install`

#### Run App

`npm start`

#### Build App

`npm run build`

#### Run Test

`npm test`
